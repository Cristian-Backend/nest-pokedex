# Ejecutar en desarrollo

1. clonar el repositorio
2. ejecutar

```
npm install 
```

3. tenes nest CLI instalado 

```
nest i -g @nestjs/cli
```

4. levantar la base de datos
```
docker-compose up -d
```

## Stack usado

- MongoDB

- NestJS
